Image Non-blind Deconvolution

[Click to download a package with the executable and examples]

This is a free executable for solving the non-blind deconvolution problem, to facilitate the research of image deblurring. The format of the command is as follows:

robust_deconv.exe inputImageName kernelFileName outputImageName boundaryType noiseStr localPriorWeight

Parameter Definition:

inputImageName The filename of  the input blur image, in this version, .bmp, .png and .jpg files are supported
kernelFileName The filename of the kernel image
outputImageName The filename of the target image which is used to store the output result
boundaryType The boundary type of the input blur image, if boundaryType = 0, the program assumes the pixel values outside the boundary of the latent/clear image are computed by presuming the latent signal is periodic; if boundaryType = 1, the program assumes no prior knowledge about the the pixels outside the boundary of the latent image.  If boundaryType = 0 is used, there may be ringing artifacts around the image boundary. In this case, a Matlab function "edgetaper" can be used to preprocess the blur image, to reduce the ringings.
noiseStr A parameter which indicate the strength of noise, increase this parameter will make a result looks smooth, while reducing this parameter will make more local structure visible and may produce more visual artifacts, its default value is shown in "DeconvAgent.h"
localPriorWeight A parameter to adjust the smoothness enforced on flat regions, its default value is shown in "DeconvAgent.h"

System Requirement:

Windows XP, 2003 Server and Vista.
